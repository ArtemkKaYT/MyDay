# MyDay — Приложение для сводки дня

Приложение для быстрой сводки вашего дня с информацией о погоде, расписании и личных заметках. Разработано на Python с использованием PyQt6.

## 🎯 Возможности

- 🌦️ **Прогноз погоды** — текущая погода
- 📅 **Расписание** — управление событиями
- 📝 **Заметки** — создание и редактирование личных заметок
- 📊 **Краткая сводка** — быстрый обзор всей информации на одном экране
- 🎨 **Тематическое оформление** — адаптивный дизайн

## 📋 Требования

- **Python** 3.13+
- **ОС**: Windows

## 🚀 Установка

### Вариант 1: Использование готового .exe файла

1. Перейдите в папку `dist/`
2. Запустите `main.exe`

### Вариант 2: Использование исходного кода

1. Установите необходимые зависимости:
Например:
```bash
pip install PyQt6 requests python-dotenv
```

2. Запустите приложение:
```bash
python main.py
```

> **Примечание**: Для работы прогноза погоды необходимо вставить api в файл `.env`

## ⚙️ Конфигурация

### Получение API ключа для погоды

1. Перейдите на [https://openweathermap.org/api](https://openweathermap.org/api)
2. Зарегистрируйтесь (бесплатный план достаточен)
3. Получите ваш **API Key**
4. Добавьте его в файл `.env`:

```env
WEATHER_API_KEY=your_api_key_here
CITY=Saint Petersburg
```

**Поддерживаемые города:**
- Любой город мира (используется название на английском)
- По умолчанию: `Saint Petersburg`

## 🎮 Использование

### Запуск приложения
```bash
python main.py
```

### Управление заметками
1. Перейдите на вкладку "Заметки"
2. Нажмите на кнопку добавления новой заметки
3. Введите текст и сохраните

### Управление расписанием
1. Перейдите на вкладку "Расписание"
2. Добавьте новое событие
3. Укажите дату и время

### Просмотр погоды
1. Убедитесь, что API ключ настроен в `.env`
2. Информация о погоде отобразится автоматически

## 🔄 Сборка .exe файла

Если вы хотите пересобрать приложение в исполняемый файл:

```bash
pip install pyinstaller
pyinstaller main.spec
```

Готовый файл будет в папке `dist/`

## 🛠️ Технический стек

- **PyQt6** — графический интерфейс
- **OpenWeatherMap API** — данные о погоде
- **JSON** — хранение данных (заметки, расписание)
- **Python 3.13** — язык программирования

## 📝 Примечания

- Все данные хранятся локально в JSON файлах
- При первом запуске создаются стандартные файлы данных
- Приложение работает полностью оффлайн, кроме функции прогноза погоды
- Интернет требуется только для загрузки данных о погоде
